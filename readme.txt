== Description ==

jMarquee makes it easy to add horizonatlly or vertically scrolling text or images to posts.


== Installation ==

1. Upload the whole `jmarquee` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
